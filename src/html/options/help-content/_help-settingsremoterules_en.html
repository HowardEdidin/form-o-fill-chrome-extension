Using remote rules setup by others

If you want to use Form-O-Fill but do not want to write your own rules you can use rules written by others (eg. colleagues).
These are called remote rules.
The only condition is that these rules must be available via http or https.
This is how you activate remote rules:

Step 1:  Have someone build the rules and workflows that you need locally inside Form-O-Fill.
Step 2:  Export those rules a .js file
Step 3:  Put them on a server and make them accessible via http or https
Step 4:  Take the URL to the .js file and put it in the setting for "Activate remote rules import"
Step 5:  Press and Form-O-Fill will report success or failure.

Form-O-Fill will keep these rules in a separate storage and will auto-update those rules every ##Utils.alarmIntervalInMinutes## minutes.

Those rules will then also be used when searching for rules and workflow matches.

Using the on-click rules installer

To make using remote rules even easier, you can use http://form-o-fill.github.io/import-remote-rules/ to generate a link that will auto-install your remote rules.

This makes it easier to pass on those remote rules and make them usable for users that might not find it easy to follow the steps mentioned above.

Visiting the generated link will import the remote rules and display a notification in the top right corner with success or failure.
